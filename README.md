# Delhivery Network Analytics Dashboard

This repository contains the Streamlit dashboard developed for visualizing and analyzing logistics network performance. The dashboard provides interactive insights into bottleneck hubs, SLA breach corridors, risk scoring, network connectivity, and operational KPIs using preprocessed project outputs.

**Tech Stack:** Streamlit, Pandas, Plotly, NetworkX

**Note:** This repository contains only the dashboard application and supporting visualization files. The complete project pipeline, model development, and data processing code are not included.
